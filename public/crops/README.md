# Crop Images

Place your crop images in this folder with the following names:
- rice.jpg
- wheat.jpg
- maize.jpg
- pulses.jpg
- sugarcane.jpg
- cotton.jpg
- groundnut.jpg
- mustard.jpg
- soybean.jpg
- oilseeds.jpg
- jute.jpg
- bajra.jpg
- jowar.jpg
- ragi.jpg
- default-crop.jpg

Image requirements:
- Recommended size: 400x400 pixels
- Format: JPG/JPEG
- File size: Optimize for web (< 100KB per image)
- Aspect ratio: 1:1 (square)

Until you add the actual images, the component will show the default-crop.jpg as a fallback.
